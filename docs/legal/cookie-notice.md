# Cookie and Tracking Notice

## 1. Current Setup

Zemio currently intends to use only technically necessary cookies and comparable local storage mechanisms required for:

- authentication and session handling;
- security-related request handling;
- user interface preferences necessary for the requested service.

Zemio does not intend to use analytics, advertising, cross-site tracking, or session replay in production at this time.

## 2. Legal Basis

Technically necessary storage or access on a user's device is used only where required to provide the requested digital service or to ensure security and core functionality.

Processing of personal data that occurs in connection with such technically necessary features is described in the Privacy Policy.

## 3. No Non-Essential Tracking at Present

At the date of this draft, Zemio does not intend to set non-essential cookies or similar tracking technologies requiring prior consent.

If Zemio introduces optional analytics, session replay, or other non-essential tracking in the future, Zemio will:

- update this notice;
- implement an appropriate consent mechanism where legally required;
- activate such technologies only after the necessary legal and technical steps are completed.
