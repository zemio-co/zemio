# Subprocessor List

This list is a working draft of the main subprocessors and infrastructure providers intended for Zemio production use.

## 1. Vercel

Purpose:

- application hosting;
- deployment infrastructure;
- edge and platform-related operational services.

Location note:

- production region and related transfer configuration should be confirmed before launch.

## 2. Neon

Purpose:

- managed PostgreSQL database services.

Location note:

- intended EU-region configuration should be confirmed and documented in the customer onboarding and vendor register.

## 3. Hetzner

Purpose:

- object storage for attachments and related file content.

Location note:

- intended storage region should be confirmed before launch.

## 4. Resend

Purpose:

- operational email delivery, including invitations and system-triggered notifications.

## 5. Microsoft

Purpose:

- identity authentication through Microsoft login.

## 6. Better Stack

Purpose:

- operational monitoring and error tracking.

## 7. Notes

- This list should be reviewed and updated before production launch.
- The final production version should include links to the current vendor privacy and subprocessor documentation and should identify the relevant service region where available.
- If additional subprocessors are introduced, Zemio should notify customers in line with the Data Processing Agreement.
